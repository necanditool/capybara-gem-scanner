=========================================================
  Capybara Gem Scanner  -  Community Edition (v51)
  PATCHNOTES  (Deutsch)
=========================================================

START (ein Klick)
  - Alle Dateien in EINEM Ordner lassen.
  - Doppelklick auf "Edelstein-Scanner starten.bat".
  - Erster Start installiert alles automatisch (kurz warten).
  - Falls Python frisch installiert wurde: die .bat NOCH EINMAL doppelklicken.

BEDIENUNG (kurz)
  - Oben rechts: Sprache (DE <-> EN) und der Hell/Dunkel-Schalter (Mond/Sonne).
    Darunter den Build waehlen - das aendert die Bewertung in allen Tabs.
  - Schnell-Scan: einen Bereich markieren -> liest ALLE Steine, bewertet sie
    fuer deinen Build und zeigt oben den Build-Doktor (Behalten / Tauschen).
  - Slot: PvE/PvP und Slot waehlen, zwei Bereiche festlegen (Ausgeruestet
    rechts, Auswahl links), Stein anwaehlen, "Scannen & vergleichen" -> Urteil.
  - Held: zeigt deine Ausruestung gegen den Build. Der Build-Check vergleicht
    deine gescannten Steine mit dem Build und markiert, was nicht passt.
  - Skills: Bereich der angebotenen Skills markieren, "Skills pruefen" ->
    das Tool sagt pro Build, was du LERNEN und was du MEIDEN solltest.
  - Loadout: alle Slots auf einen Blick. Guide: komplette Build-Specs.

---------------------------------------------------------
  ANTIVIRUS-HINWEIS (Fehlalarm)
---------------------------------------------------------
  Manche Virenscanner schlagen evtl. an. Das ist ein FALSCH-ALARM, keine
  Schadsoftware. Grund: Das Tool macht Bildschirmfotos (um die Edelsteine zu
  lesen) und die Start-Datei (.bat) installiert beim ersten Start automatisch
  Python-Pakete - dieses Verhalten melden Scanner vorsichtshalber.
  Es werden KEINE Daten gesendet (nur das optionale Online-Update laedt von
  einer URL, die DU selbst eingibst). Der Quellcode "capybara_gem_scanner.py"
  ist offen und lesbar.
  Loesung: den Tool-Ordner als Ausnahme/Whitelist hinzufuegen. Windows
  Defender -> "Viren- & Bedrohungsschutz" -> "Einstellungen verwalten" ->
  "Ausschluesse" -> "Ausschluss hinzufuegen" -> Ordner. (Oder nach einer
  Blockierung im Schutzverlauf auf "Aktion zulassen" klicken.)
